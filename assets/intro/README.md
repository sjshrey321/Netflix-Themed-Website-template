# Intro Video

Drop your Netflix-style intro video here as **`intro.mp4`**.

The site will automatically detect and play it on boot. If the file is missing, the animated `NETFLIX` logo fallback is used instead.

## Recommended specs

- **Filename:** `intro.mp4` (exact name required)
- **Format:** MP4 (H.264 codec for best browser compatibility)
- **Duration:** 3–5 seconds
- **Resolution:** 1920×1080 or higher
- **Audio:** Included (the iconic "ta-dum" sound)

## Optional alternates

If you want browser fallback support, you can also drop:
- `intro.webm` — better compression for modern browsers
- `intro.mov` — fallback for Safari quirks
