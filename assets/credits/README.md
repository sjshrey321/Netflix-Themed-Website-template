# Credits Page Assets

Drop these in here for the credits page:

| Filename | Purpose |
|---|---|
| `credits-bg.jpg` | Dimmed background photo (e.g., a cozy room — the moody backdrop behind the credits) |
| `credits-clip.mp4` | Small looping video shown in the lower-left corner (a candid clip of you two) |

## Specs

- **`credits-bg.jpg`** — Landscape (16:9), at least 1920×1080. The page will dim it automatically.
- **`credits-clip.mp4`** — Square or portrait, ~5–10 seconds, looping. Plays muted by default.
